API 调用结构

把提示词拆成稳定字段，方便做批量生成、审核和版本回溯。

1 选择场景

从分类、来源和画面目标开始，先确认输出用途。

2 填入变量

替换主体、风格、构图、光线、比例、文字约束。

3 调用生成

把最终提示词传给图像 API，并记录版本与参数。

4 复盘沉淀

保存成自己的可复用模板，不再从零试错。

const response = await client.images.generate({
  model: "gpt-image-2",
  prompt: buildPrompt({
    useCase: "电商主图",
    subject: "一款透明玻璃香水瓶",
    style: "高端杂志棚拍",
    composition: "正面构图，留出电商标题区域",
    constraints: "无水印，文字必须清晰"
  }),
  size: "1536x1024",
  quality: "high"
